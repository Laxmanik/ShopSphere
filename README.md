# ShopSphere

A production-ready full-stack e-commerce application built with Java, Spring Boot, React.js, and MySQL.

> 🚧 This project is currently under active development.

## Tech Stack

### Backend
- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Maven

### Frontend
- React.js
- Vite
- Axios

### Tools
- Git
- Docker
- Postman
