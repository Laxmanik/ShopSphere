- Spring Boot Architecture
- IoC (Inversion of Control)
- Dependency Injection
- Spring Bean
- ApplicationContext
- Spring Boot Startup Lifecycle

- HikariCP
- JDBC
- JPA
- Hibernate
- Database Normalization
- Modular Monolith Architecture

---

## Key Takeaways

- Spring Boot manages beans using the IoC Container.
- Dependency Injection reduces coupling.
- JPA is a specification.
- Hibernate is the default JPA implementation.
- HikariCP manages database connections efficiently.
- Normalize databases to reduce redundancy and improve integrity.
- Design before coding.
